| **Algorithm**      | **Big O (Time)** | **The "SysAdmin" Translation**                                                                     |
| ------------------ | ---------------- | -------------------------------------------------------------------------------------------------- |
| **Linear Search**  | $O(n)$           | Grepping a log file from start to finish.                                                          |
| **Binary Search**  | $O(\log n)$      | Querying an indexed database (Dividing the search space in half).                                  |
| **Bubble Sort**    | $O(n^2)$         | The "Broadcast Storm." Inefficiently swapping neighbors until everything settles.                  |
| **Insertion Sort** | $O(n^2)$         | Organizing a stack of physical papers one by one. (Fastest $O(n^2)$ for small/mostly sorted data). |
| **Selection Sort** | $O(n^2)$         | Finding the absolute minimum, moving it to the front, and repeating.                               |
